# Expense Tracker Project

## Description
This is a simple Expense Tracker application developed in Python. It allows users to enter multiple expenses and calculates the total amount spent.

## Features
- Enter multiple expenses
- Calculate total spending
- Simple and easy-to-use interface
- Beginner-friendly Python project

## Technologies Used
- Python

## How to Run
1. Run the Python file.
2. Enter the number of expenses.
3. Enter each expense amount.
4. View the total amount spent.
